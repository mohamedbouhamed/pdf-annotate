# PDF Annotation App (Swift)

Built as a personal project to explore PDF manipulation and UI design in Swift, showcasing my iOS development skills.

A personal iOS project built from scratch in **Swift**, enabling users to annotate PDFs with highlights, notes, and drawings. Designed with a clean and intuitive UI for seamless PDF interaction on iPhone and iPad.

## Key Features

- Open and navigate multi-page PDFs
- Highlight text, add notes, and freehand drawings
- Save or share annotated PDFs
- Fully built in **Swift** using **PDFKit** and **UIKit/SwiftUI**

## Tech Skills Demonstrated

- iOS app development (Swift, UIKit/SwiftUI)
- PDF handling and annotation
- UI/UX design for mobile applications
- File management and export features
